## Summary
Added VTT subtitle parser that downloads YouTube subtitles via yt-dlp and parses them into TranscriptSegments, skipping Whisper when subtitles are available (1700 segments in <1 sec vs 30 min Whisper). Fixed SQLite BUSY errors with busy_timeout=5000 and SetMaxOpenConns(1). Fixed yt-dlp subtitle download (ignore exit code on partial language failure), video format (--merge-output-format mp4), ffmpeg (-y flag + stderr logging), and whisper sidecar (missing requests dependency). Switched to HTTP transport for Docker Desktop Mac compatibility.
