## Summary
Dynamic frame extraction now calculates minimum frames based on video duration using ffprobe, so longer videos get proportionally more frames (e.g., 122 instead of 10 for a 10-min video). Chunk duration reduced from 45s to 30s for finer granularity. Summarization prompt rewritten to preserve the speaker's voice instead of producing third-person summaries, with configurable compression ratio and dynamic target word count. First end-to-end analysis report added as test output.
